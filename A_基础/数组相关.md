#### 一、数组删除元素

* 使用 arr.splice(index,howMany,item1,...,itemx) 方法
* 从index下标开始删除 删除howMany个元素，并且在删除位置添加 item1,...,itemx 新元素
* 此方法返回值是删除掉元素的新数组
* 此方法会改变原始数组，所以不用重新给原始数组赋值

``` JavaScript
var arr = ['item1','item2','item3'];
console.log(arr);

var index = 1;//从第几个（下标）开始删除
var howMany = 1;//删除几个
arr.splice(index,howMany);

console.log(arr);
```

``` text
输出结果：
['item1','item2','item3']
['item1','item3']
```
